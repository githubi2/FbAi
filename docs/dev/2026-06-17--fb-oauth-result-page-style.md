# FB OAuth 结果页样式统一

**日期**: 2026-06-17

## 修改内容

### 后端 — `art-design-server/handlers/fb_handler.go`

修改了两处内联 HTML 页面，使其样式与后台 `ArtResultPage` 结果页组件保持一致：

#### 1. `Callback()` — 授权成功页（line 118-215）
- **之前**: 紫色渐变背景 + 白色卡片 + 浅绿圆形图标 + 小号文字
- **之后**: 浅灰背景 `#f5f7fa` + 绿色实心圆形图标（`#19BE6B`, 88px）+ 深色标题（`#323251`, 30px）+ 灰色说明文字 + 灰底信息框

#### 2. `ShortRedirect()` — 链接过期页（line 258-360）
- **之前**: 粉色背景 + 简单文字 + ⚠ emoji
- **之后**: 浅灰背景 + 红色实心圆形图标（`#ED4014`, 88px）+ 深色标题 + 灰色说明 + 灰底提示框
